## Issue 1: Missing Image Asset
**File:** `src/Redball.UI.WPF/Views/SettingsWindow.xaml:212`
**Severity:** Medium

**Problem:**
SettingsWindow references `/Assets/redball.png` but only `redball.ico` is included in the project.

**Current Code:**
```xml
<Image Source="/Assets/redball.png" Width="64" Height="64" Margin="0,0,0,16"/>
```

**Fix:**
Either:
1. Add `redball.png` to the Assets folder and include in csproj:
   ```xml
   <Content Include="Assets\redball.png" CopyToOutputDirectory="PreserveNewest" />
   ```
2. Or change the reference to use the existing ICO file:
   ```xml
   <Image Source="/Assets/redball.ico" Width="64" Height="64" Margin="0,0,0,16"/>
   ```

## Issue 2: Multiple Window Instances
**File:** `src/Redball.UI.WPF/Views/MainWindow.xaml.cs:144,153`
**Severity:** Medium

**Problem:**
ShowSettings() and ShowAbout() create new windows every time without checking if one is already open. Users can open unlimited duplicate windows.

**Current Code:**
```csharp
public void ShowSettings()
{
    Dispatcher.Invoke(() =>
    {
        var settings = new SettingsWindow();
        settings.Show();
    });
}
```

**Fix:**
Track window instances and activate existing windows:
```csharp
private SettingsWindow? _settingsWindow;
private AboutWindow? _aboutWindow;

public void ShowSettings()
{
    Dispatcher.Invoke(() =>
    {
        if (_settingsWindow != null && _settingsWindow.IsLoaded)
        {
            _settingsWindow.Activate();
            _settingsWindow.Focus();
            return;
        }
        _settingsWindow = new SettingsWindow();
        _settingsWindow.Closed += (s, e) => _settingsWindow = null;
        _settingsWindow.Show();
    });
}

public void ShowAbout()
{
    Dispatcher.Invoke(() =>
    {
        if (_aboutWindow != null && _aboutWindow.IsLoaded)
        {
            _aboutWindow.Activate();
            _aboutWindow.Focus();
            return;
        }
        _aboutWindow = new AboutWindow();
        _aboutWindow.Closed += (s, e) => _aboutWindow = null;
        _aboutWindow.Show();
    });
}
```

## Issue 3: Empty Catch Blocks
**File:** `src/Redball.UI.WPF/Views/MainWindow.xaml.cs:27`
**Severity:** Low

**Problem:**
Silent failure in Log method - errors are swallowed with no diagnostic output.

**Current Code:**
```csharp
private static void Log(string message)
{
    try
    {
        var timestamp = DateTime.Now.ToString("yyyy-MM-dd HH:mm:ss");
        File.AppendAllText(LogPath, $"[{timestamp}] {message}{Environment.NewLine}");
    }
    catch { }
}
```

**Fix:**
```csharp
private static void Log(string message)
{
    try
    {
        var timestamp = DateTime.Now.ToString("yyyy-MM-dd HH:mm:ss");
        File.AppendAllText(LogPath, $"[{timestamp}] {message}{Environment.NewLine}");
    }
    catch (Exception ex)
    {
        System.Diagnostics.Debug.WriteLine($"Log failed: {ex.Message}");
    }
}
```

## Issue 4: Null Reference After Dispose
**File:** `src/Redball.UI.WPF/Views/MainWindow.xaml.cs:162`
**Severity:** Low

**Problem:**
_trayIcon is disposed but not set to null, risking ObjectDisposedException on subsequent access.

**Current Code:**
```csharp
public void ExitApplication()
{
    if (_trayIcon != null)
    {
        _trayIcon.Dispose();
    }
    Application.Current.Shutdown();
}
```

**Fix:**
```csharp
public void ExitApplication()
{
    if (_trayIcon != null)
    {
        _trayIcon.Dispose();
        _trayIcon = null;
    }
    Application.Current.Shutdown();
}
```

## Issue 5: IPC Resource Leak
**File:** `src/Redball.UI.WPF/App.xaml.cs:200`
**Severity:** Medium

**Problem:**
OnExit only disposes _pipeServer but _pipeReader and _pipeWriter are not disposed, causing resource leak.

**Current Code:**
```csharp
protected override void OnExit(ExitEventArgs e)
{
    Log($"=== Redball.UI.WPF Exiting (code: {e.ApplicationExitCode}) ===");
    _pipeServer?.Dispose();
    base.OnExit(e);
}
```

**Fix:**
```csharp
protected override void OnExit(ExitEventArgs e)
{
    Log($"=== Redball.UI.WPF Exiting (code: {e.ApplicationExitCode}) ===");
    _pipeReader?.Dispose();
    _pipeWriter?.Dispose();
    _pipeServer?.Dispose();
    base.OnExit(e);
}
```
