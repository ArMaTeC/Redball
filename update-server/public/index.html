How to Install

Choose your preferred installation method. Each option includes step-by-step instructions.

NSIS Installer (Recommended)

  1. Download the Redball-Setup.exe file from the download section above
  2. Double-click the downloaded file to launch the installer
  3. If Windows SmartScreen appears, click "More info" then "Run anyway"
  4. Follow the setup wizard prompts - choose your features (shortcuts, auto-start, .NET runtime)
  5. Click "Launch Redball" on the finish page to start using it immediately
Administrator Required?

No admin rights needed for per-user installation. Admin only required for all-users install.

Portable ZIP

  1. Download the Redball.zip file from the download section
  2. Extract the ZIP to any folder (e.g., C:\Tools\Redball or your Desktop)
  3. Run Redball.UI.WPF.exe to start the application
  4. No installation required - runs directly from the extracted folder
# Using PowerShell to extract and run: Expand-Archive -Path Redball.zip -DestinationPath C:\Tools\Redball C:\Tools\Redball\Redball.UI.WPF.exe
Perfect for:

Work computers without admin rights, USB drives, or quick testing before full install.

Windows Package Manager (winget)

  1. Open PowerShell or Command Prompt as Administrator
  2. Run the install command below
  3. Wait for the installation to complete
  4. Redball will be available in your Start Menu
# Install via winget: winget install ArMaTeC.Redball # Upgrade to latest version: winget upgrade ArMaTeC.Redball # Uninstall: winget uninstall ArMaTeC.Redball
Automatic Updates

Winget makes it easy to keep Redball updated. Just run winget upgrade periodically.

Frequently Asked Questions

Quick answers to common questions about Redball.

Is Redball really free?
Yes! Redball is completely free and open source under the MIT License. No ads, no subscriptions, no hidden costs. You can even view and modify the source code on GitHub.
How does Redball keep my computer awake?
Redball uses the Windows SetThreadExecutionState API to tell Windows that the system is in use, preventing it from going to sleep. It can also optionally send invisible F15 keypresses (heartbeat) to prevent application-level idle detection.
Does Redball work with Windows 11?
Yes, Redball is fully compatible with Windows 10 and Windows 11 (both Home and Pro editions). It also works on Windows Server 2019 and 2022.
What is TypeThing and how does it work?
TypeThing is a clipboard typer that simulates human-like typing. Press Ctrl+Shift+V anywhere and Redball will type your clipboard contents with natural delays between keystrokes. Perfect for pasting into remote desktop sessions, VMs, or secure fields that block Ctrl+V.
Will Redball slow down my computer?
No. Redball is designed to be lightweight, using less than 50MB of RAM and minimal CPU. The keep-awake check happens every 59 seconds with virtually no performance impact. You can also enable battery-aware mode to automatically pause when on battery power.
How do I update Redball?
Redball can check for updates automatically (optional). When an update is available, you'll get a notification. Click it to download and install the update automatically. If you installed via winget, use winget upgrade ArMaTeC.Redball.
Where is the source code?
Redball is open source and hosted on GitHub at github.com/ArMaTeC/Redball. Feel free to report issues, suggest features, or contribute pull requests!
