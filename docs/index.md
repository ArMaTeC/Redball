# Redball API Documentation

Welcome to the Redball API documentation. This site provides detailed information about the classes, interfaces, and methods available in the Redball.UI.WPF assembly.

## Getting Started

Redball is a system tray utility to prevent Windows from going to sleep. Browse the API reference to understand the internal architecture and extensibility points.

## Namespaces

- **Redball.UI.Services** - Core services (KeepAwake, Config, Analytics, etc.)
- **Redball.UI.ViewModels** - MVVM view models
- **Redball.UI.Views** - WPF views and windows
- **Redball.UI.Models** - Data models and configuration
