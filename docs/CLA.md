## Sign-Off Procedure

All Contributions must include a sign-off in the commit message indicating You accept this Agreement:

```text
Signed-off-by: Your Name <your.email@example.com>
```

You can add this automatically using Git:

```bash
git commit -s -m "Your commit message"
```

By submitting a Pull Request or other Contribution to the Project, You indicate Your acceptance of this Agreement.
